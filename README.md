# Mouse Follower

An interactive demo built with **React**, **Vite**, and **Tailwind CSS**.

---

## Demo

[nataliatorres.github.io/mouse-follower](https://nataliatorres.github.io/mouse-follower/)

---

## Features

- Start/stop following functionality
- Toggle the component on and off
- Built to demonstrate `useEffect` and `useState` hooks

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/nataliatorres/mouse-follower.git
   cd mouse-follower
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
